出力ルール

  • 画像は n×n に分割されます。
  • 連番は 左上 → 右下、行優先で振られます。
  • 元のファイル名を使って sample_001.png の形式で保存します。
  • 最後の行・列は、端数がある場合も画像の右端・下端まで含めて切り出します。
  • PDF は 1ページに1枚 で出力されます。貼り合わせ印刷向けです。
